 BabbageFaces comprime o tamanho das respostas AJAX geradas pelas aplicações baseadas no Mojarra 2.2+ e PrimeFaces 5+. Clientes que estejam sofrendo com problemas de lentidão de conexão irão notar o efeito. A idéia geral é que você não precisa substituir o HTML inteiro para obter pequenas mudanças. Mais do que isso, não é tarefa do desenvolvedor identificar e realizar tais mudanças. Deixe isso para o BabbageFaces. Ele conhece alguns truques que você mesmo teria muita dificuldade para realizar.

BootsFaces combina uma versão sólida e robusta de JSF com a beleza e o visual do Bootstrap. Ele pode ser usado separadamente (stand-alone), mas é também um complemento natural para o AngularFaces e PrimeFaces. A página que você está vendo é uma página JSF rodando em BootsFaces, AngularFaces e OmniFaces.
