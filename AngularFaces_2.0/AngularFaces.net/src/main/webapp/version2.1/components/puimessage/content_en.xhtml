pui-message (client-side widget)

<puimessage> is an AngularJS widget. It displays an error message if the input field value violates a validation rule. In most cases, you haven't got anything to do with <puimessage>. It's an internal widget automatically generated by AngularFaces.

Attributes of <pui-message>
Attribute Default value Description
for (none) id of the input field the label belongs to.
primefaces false If true, the label is rendered just like a PrimeFaces <prime:outputLabel />.
