pui-label (client-side widget)

<puilabel> is an AngularJS widget. It displays a label that becomes red if a validation rule is violated. In most cases, you haven't got anything to do with <puilabel>. It's an internal widget automatically generated by AngularFaces.

Attributes of <pui-label>
Attribute Default value Description
for (none) id of the input field the label belongs to.
primefaces false If true, the label is rendered just like a PrimeFaces <prime:outputLabel />.
bootsfaces false If true, the label is rendered just like a Bootstrap label.
label (none) The text of the label.
