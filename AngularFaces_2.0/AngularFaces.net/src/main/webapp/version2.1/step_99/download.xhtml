Download

The stable releases of AngularFaces are available at Maven central. You can either download the jar manually by opening http://search.maven.org/#search%7Cga%7C1%7Cbeyondjava in your browser, or by adding the dependency to your pom.xml or .gradle build file:

If you're the adventurous type, you can also build AngularFaces from source. But be warned: Every once in a while the head revision on GitHub will be broken. How to build the project from source is described in the Bootstrapping chapter of this tutorial.
