Editable tables with JSF beans

Starting with AngularFaces 2.1.8, it's possible to use editable tables with an JSF bean. You can add, edit and remove rows.

Unfortunately, the code is a bit cumbersome. That's due to type erasure. It's not possible to convert a Json object containing an array to a Java object unless you know the type. AngularFaces doesn't know your objects, so it needs a little help in the Java bean.
