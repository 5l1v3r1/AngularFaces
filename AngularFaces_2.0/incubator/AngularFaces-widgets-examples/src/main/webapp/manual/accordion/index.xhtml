Based firmly in the future... The pui:accordion is a vertically stacked list of items, such as labels or thumbnails. Each item can be "expanded" or "stretched" to reveal the content associated with that item. There can be zero expanded items, exactly one, or more than one items expanded at a time, depending on the configuration. One or two source codes. pui:accordion has these properties:
k-options a JSON object configuring the accordion (see Kendo UI Panelbar API)
expandMode possible values:
  • "single": expanding an accordion bar closes the other bar.
  • "multiple": default.
ng-model The angular model.
value The JSF bean.
style CSS inline styles.
styleClass CSS class.
Now the content of the panel is visible.
