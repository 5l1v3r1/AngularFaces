Using AngularJS mustaches in a JSF page

Traditionally JSF pages are rendered on the server side. Each update requires an expensive AJAX request (or an even more expensive HTML request). AngularJS pages, in contrast, are rendered on the client side. Changes are rendered extremely fast, but there's no connection to the back-end. AngularFaces spans a bridge over this gap, making JSF bean attributes available to AngularJS and synchronizing parts of the AngularJS model with the server's JSF beans.
